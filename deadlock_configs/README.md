# Deadlock Configs

A collection of configs for Deadlock. Original configs by [@Leif](https://www.youtube.com/@leifdeadlock).

# How to Run

Copy the files to `..\Deadlock\game\citadel\cfg` folder.

In game run the following commands in the dev console (by default key `F7`)

```
sv_cheats 1
exec leiftools
```
